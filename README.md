# Jogo de Escolhas

Feito por Rafael Alves Faria.

Primeiro código feito por mim na linguagem C, um simples jogo de onde o jogador
deve escolher uma das opções que aparecerem na tela para continuar.


[![Meu Vídeo](https://img.youtube.com/vi/HniYqxrEWM4/mqdefault.jpg)](https://www.youtube.com/watch?v=HniYqxrEWM4)

## Como jogar?

Ao iniciar o programa em um compilador de C irá aparecer uma mensagem sobre a história do jogo, onde o usuário porderá escolher entre as opções A ou B. A escolha do usuário influencia na história e no final que você conseguirá.

No total são 3 finais diferentes, com duas perguntas para finalizar o jogo.

Ao finalizar o jogo com o final verdadeiro, será pedido para que o usuário dê uma nota para o jogo, dependendo da nota dada uma mensagem irá ser printada no terminal.

# Limitações:
- Se digitar um texto que comece com A ou B, ele irá idenrificar como se fosse as letras;
- Em alguns compiladores pode não funcionar corretamente (Testados: GDB Online);
- Funciona somente no terminal;
